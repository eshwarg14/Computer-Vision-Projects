# Computer-Vision-Projects
A collection of real-time AI computer vision projects built with Python &amp; OpenCV - includes emotion detection, face recognition, gesture control, pose estimation, object recognition, drowsiness detection and more.

---

## 📁 Repository Structure

```
AI-Computer-Vision-Projects/
├── project/
│   ├── main.py
│   ├── requirements.txt
│   └── README.md
└── ...
```

> Each project has its own folder with its own README and requirements file.

---

## 🛠️ Tech Stack

- Python 3.x
- OpenCV
- MediaPipe
- TensorFlow / scikit-learn
- Arduino (for hardware-integrated projects)

---

## 🚀 Getting Started

1. Navigate to any project folder
   cd emotion-recognition

2. Install dependencies
   pip install -r requirements.txt

3. Run the project
   python main.py

---

## 📌 Requirements

- Python 3.8 or above
- Webcam (for real-time projects)
- Arduino + required sensors (for hardware projects)
